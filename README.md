# Blog Repository

This repository contains a blog application with separate frontend and backend folders.

## Structure

- `Backend/` - Node.js backend application
- `Frontend/` - Frontend application files

## Backend Setup

1. Open a terminal in `Backend/`
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Notes

- The backend folder includes a local `.gitignore` to ignore `node_modules`, `.env`, and build output.
